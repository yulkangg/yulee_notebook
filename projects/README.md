# Projects

Use this folder for the active research work.

Suggested organization:

- `matlab/`: MATLAB scripts, functions, live scripts, and project files
- `python/`: Python scripts, notebooks, packages, and environment notes
- `papers/`: article drafts, paper notes, and literature summaries
- `research-notes/`: dated research notes, meeting notes, and progress logs
- `presentations/`: slides, posters, and talk materials
- `data/`: small shareable datasets or data dictionaries

For larger projects, create one subfolder per project inside the relevant area.
