# Research Notes

Use this folder for dated notes from research sessions, meetings, readings, and experiments.

Recommended file names:

```text
2026-06-01-meeting-notes.md
2026-06-03-literature-notes.md
2026-06-07-experiment-log.md
```

A good note usually records:

- what was attempted
- what worked
- what failed or was confusing
- important links, equations, references, or code files
- next steps before the next meeting
