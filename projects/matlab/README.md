# MATLAB

Place MATLAB research code here.

Suggested practices:

- Put reusable functions in clearly named `.m` files.
- Keep exploratory scripts separate from polished analysis scripts.
- Add comments explaining inputs, outputs, and assumptions.
- Avoid committing large generated output files unless your mentor asks you to keep them here.

Possible structure for a larger MATLAB project:

```text
project-name/
  code/
  figures/
  results/
  README.md
```
