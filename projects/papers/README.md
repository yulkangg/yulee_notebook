# Papers

Use this folder for drafts, outlines, literature notes, and paper summaries.

Good contents for this folder include:

- article outlines
- manuscript drafts
- annotated bibliographies
- short summaries of papers read for the project
- notes on methods, definitions, or results from the literature

Before uploading published PDFs, check copyright and course/research group expectations. Often it is better to store citations and summaries here rather than full copyrighted articles.
