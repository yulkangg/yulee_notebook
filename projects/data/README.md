# Data

Use this folder only for small, shareable datasets and data documentation.

Before committing data, check with your mentor. Do not upload restricted, private, proprietary, or very large data files.

Useful files for this folder include:

- `data-dictionary.md`: variable names and definitions
- `sources.md`: where the data came from
- small example datasets used in tutorials or tests

For large datasets, store instructions here that explain where the data lives and how approved collaborators can access it.
