# Student Guide

This repository is a template for organizing undergraduate research materials.

## First Steps

1. Create your own repository from this template using `Use this template`.
2. Edit the new repository name so it clearly identifies you or your project.
3. Add your profile materials in `profile/`.
4. Add project files in `projects/`.
5. Keep notes current in `projects/research-notes/`.

## What To Commit

Good files to commit:

- Markdown notes
- source code
- small example datasets
- figures and plots that are useful for reports
- presentation drafts
- project documentation

Check with your mentor before committing:

- unpublished manuscripts
- large data files
- private or restricted datasets
- copyrighted PDFs
- files containing personal information

## Naming Files

Use descriptive names. For notes, begin with the date in `YYYY-MM-DD` format.

Examples:

```text
2026-06-01-meeting-notes.md
2026-06-03-first-simulation-results.md
2026-06-10-paper-summary-smith-2024.md
```

## Keeping The Repository Useful

A good research repository should help another person understand what you did, how you did it, and what should happen next. When in doubt, add a short README or note explaining the purpose of a folder or file.
