# GitHub Template Setup

For the repository owner:

1. Open the repository on GitHub.
2. Go to `Settings`.
3. In `General`, check `Template repository`.
4. Return to the repository home page and confirm that `Use this template` appears.

For students:

1. Click `Use this template`.
2. Choose `Create a new repository`.
3. Name the repository clearly.
4. Choose public or private visibility according to the mentor's instructions.
5. Create the repository and begin replacing placeholder guidance with project materials.

Suggested repository naming patterns:

```text
firstname-lastname-research
lastname-reu-2026
project-short-name
```
